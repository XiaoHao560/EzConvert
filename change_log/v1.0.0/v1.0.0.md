## [v1.0.0] 2026-07-01

EzConvert first release v1.0.0! 🎉
简转 (EzConvert) 第一个 release v1.0.0 正式版本！🎉

### Added

- Added display of current version changelog on first launch after app update
- Added real-time log refresh mechanism to the Log Viewer interface
- Added tooltips for play/fullscreen buttons in the Preview interface
- Added application crash screen
- Added custom FFmpeg task parameters
- Added language settings
- Added English language support

### Fixed

- Fixed lag issue when expanding log cards with large log volumes in the Log Viewer interface

### Optimized

- Optimized Log Viewer log cards to fixed height with visible-item-only rendering, improving card expansion speed

### Changed

- Redesigned UI/UX for Video/Audio Tasks
- Refactored FFmpeg task execution logic to use WorkManager for running FFmpeg tasks
- Refactored text usage to reference strings via string resources

### Removed

- Removed FFmpegKit 16kb support (issue #166)

---

### 新增
- 新增 更新应用后，首次打开时显示当前版本更新日志
- 新增 `日志查看界面` 日志实时刷新机制
- 新增 `预览界面` 播放按钮/全屏按钮 tooltip
- 新增 应用崩溃界面
- 新增 自定义FFmpeg任务参数
- 新增 语言设置
- 新增 英语语言

### 修复
- 修复 `日志查看界面` 在日志量较大时展开日志卡片会发生卡顿的问题

### 优化
- 优化 `日志查看界面` 日志查看卡片为固定高度，仅渲染可见项，提升卡片展开速度

### 变更
- 重构 `视频/音频 任务` UI/UX
- 重构 FFmpeg 任务执行逻辑，使用 WorkManager 执行 FFmpeg 任务
- 重构 文字使用，使用字符串的方式引用文字

### 移除
- 移除 FFmpegKit 16kb 支持 (issue #166)
